The following tags are used to organise the HTML document into structured sections

## SECTIONING TAGS
```html
<header> - often contains the website branding or logo

<nav> - navigation links of a section or the web page

<footer> - footer of a content section or the webpage, if often contains secondary links, copyright notice, privacy policy and cookie policy links

<main> - specifies the main content of a section or the web page

<aside> - secondary set of content that us not required to understand the main content, usually add go on this

<article> - an independent, self-contained block of content such as blog post or product

<section> - a standalone section of the document that is often used with body and article elements

<details> - A collapsed section of content that can be expanded if the user wisher to view it

<summary> - specifies the summary or caption of a <details> element
```

## CONTENT TAGS - BLOCK LEVEL TAGS
```html
<blockquote> - used to describe a quotation
    
<dd> - used to define a description for the <dt> element

<dl> - used to define a description list

<dt> - used to describe terms inside <dl> element

<figure> - applies markup to a photo image

<figcaption> - defines a caption for a photo image

<hr> - adds a horizontal line to the parent element

<li> - list items

<menu> - a semantic alternative to <ul> tag

<ol> - ordered list

<ul> - unordered list

<p> - Defines a paragraph

<pre> - used to represent a preformatted text, typically rendered using a monospace font
```

## CONTENT TAGS - INLINE TAGS
```html
<a>  - anchor link to another HTML document | Hypertext

<abbr> - specifies that the containing text is an abbrevation or acronym

<b>  - bolds the containing text, when used to indicate importance use <strong>

<br> - adds a line break

<cite> - Defines the creative work(for example a book, poem, song, movie, painting or sculpture). The text is usually rendered in italics

<code> - indicates that the containing text is a block of computer code

<data> - Indicates machine-readable data

<em> - emphasizes the contianing text

<i> - italacise the text, used to indicate idiomatic text or technical terms

<mark> - containing text should be marked or highlighted

<q> - short quotation

<samp> - used to represent a sample

<small> - used to represent small text such as copyright and legal text

<span> - A generic element for grouping content for CSS Styling

<strong> - text in bold and indicates importance

<sub> - subscript, displayed with a lowered baseline

<sup> - Containing text is superscript with raised baseline

<time> - semantic tag to display both dates and times

<u> - solid underline the text

<var> - a mathematic expression or variable
```

## EMBEDDED CONTENT AND MEDIA TAGS
```html
<audio> - Used to embed audio in web pages

<canvas> - used to render 2D and 3D graphics on web pages

<embed> - used as a container element for external content provided by an external application such as media player

<iframe> - used to embed a nested webpage

<img> - embed an images on a web page

<object> - similar to <embed> but the conent is provided by a web browser plug-in

<picture>  - An element that cotains one <img> element and more <source> elements to offer alternate images for different displays/ devices

<video>  - Embed a video on a web page

<source> - specifies media resources for <picture>, <audio> and <video> elements

<svg> - used to define SVG within a webpage
```

## TABLE TAGS
```html
<table> - defines a table element to display table data

<thead> - represents header content of a table, typically contians one <tr> element

<tbody> - main content of a table, contains one or more <tr> elements

<tfoot> - represents the footer content of a table, typically contains onre <tr> element

<tr> - represents a row in a table, contains one or more <th> when used within <thead> and one ore more <td> when used in <tbody> ot <tfoot>

<td> - Represents a cell in a table, contains the text content of the cell

<th> - defines a header cell of a table, contains text content of the header

<caption> - caption of a table element

<colgroup> - defines a semantic group of one or more columns in a table for formatting

<col> - defines a semantic column in a table
```
